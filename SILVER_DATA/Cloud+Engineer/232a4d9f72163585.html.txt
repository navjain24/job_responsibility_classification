We are seeking a talented and driven Front-End AWS Cloud Engineer with expertise in React and JavaScript to join our client's innovative engineering team. In this role, you will be responsible for designing, developing, and maintaining the front-end components of our cloud-based applications, leveraging AWS services to deliver cutting-edge user experiences. Your contributions will play a vital role in ensuring the responsiveness, performance, and scalability of our cloud-based user interfaces. This is a permanent (FTE), hybrid onsite position based in Redmond, WA. 

 
Key Responsibilities:
 
 
 Collaborate with design and back-end teams to translate user requirements and design concepts into high-quality, intuitive, and responsive front-end interfaces. 
 Develop and maintain front-end components using React, JavaScript, and/or TypeScript related libraries, ensuring optimal performance and compatibility across various devices and browsers. 
 Integrate front-end applications with back-end services and APIs, following best practices for asynchronous programming and data handling. 
 Leverage AWS services, such as S3, CloudFront, Lambda, IoT, and API Gateway, to optimize content delivery, improve performance, and enhance user experiences. 
 Work closely with UI/UX designers to implement visually appealing and user-friendly interfaces, maintaining a strong focus on aesthetics, usability, and accessibility. 
 Collaborate with mobile developers and back-end engineers to deploy and manage front-end applications in a cloud-based environment, ensuring scalability, fault tolerance, and security. 
 Monitor, analyze, and optimize front-end performance to deliver fast page load times and smooth interactions, addressing bottlenecks and issues as they arise. 
 Stay up-to-date with the latest trends and technologies in front-end development, AWS cloud services, and user experience design. 
 Participate in code reviews, technical discussions, and documentation efforts to ensure high code quality and knowledge sharing. 
 
Qualifications:
 
 
 Bachelor's degree in Computer Science, Engineering, or a related field (or equivalent practical experience). 
 Proven experience in front-end development using React and JavaScript, with a strong portfolio showcasing responsive and visually appealing user interfaces. 
 Familiarity with AWS cloud services and their integration with front-end applications, such as content delivery networks (CDNs) and serverless computing. 
 Solid understanding of web technologies, including HTML5, CSS3, and responsive design principles. 
 Knowledge of RESTful API integration and asynchronous programming techniques. 
 Strong problem-solving skills and the ability to troubleshoot complex front-end issues. 
 Excellent communication skills, both written and verbal, with the ability to collaborate effectively with cross-functional teams. 
 Familiarity with version control systems (e.g., Git) and Agile development methodologies. 
 AWS certifications and experience with cloud infrastructure provisioning (e.g., AWS Amplify) are a plus.
